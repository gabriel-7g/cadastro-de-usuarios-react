## Necessário instalado no seu computador para rodar:
- Node.JS

## Comandos para instalar no terminal:
- npm install 
- npm install axios
- npm install cors

## Comandos para Funcionar o API e ligar ao banco de dados
- cd API (Estando na pasta de cadastro de usuários)
- npm isntall cors
- npm init -y
- npm install express cors
- node server.js(Para que a API comece a rodar)

OBS: faça tudo na sequência que está para dar certo.
